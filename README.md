# 無人航空機技能証明 実地試験コース AR(拡張現実)コンテンツ
 
無人実地試験のコースをARで作成しました。  
iPhoneやiPadの画面で現実空間上にコースを映すことが出来ます。  
これを使えばメジャーを使わなくても、画面を見ながらコーンの配置やラインを引いたりしてコースを作成することが出来ます。
